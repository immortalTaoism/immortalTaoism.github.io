docker 常用操作

docker基础
docker
发布日期:   2020-05-24

1、镜像操作

操作 命令 说明
检索 docker search 关键字
eg: docker search mysql		 我们经常去docker hub上检索镜像的详细，如镜像的TAG(版本)。
拉取 docker pull 镜像名:tag :tag是可选的，tag表示标签，多为软件版本，默认是latest最新版本
列表 docker images 查看所有本地镜像
删除 docker rmi image-id 删除指定的本地镜像

2、运行镜像

docker images 查看镜像列表
docker run -d -p 8080:8080 --name mytomcat 镜像id
-d 指定容器运行与前台还是后台，默认为false
-p 指定容器暴露端口，映射一个端口

3、使用docker exec进入Docker容器

一、先使用命令docke

docker ps 查看正在运行的进程
docker exec -it 进程id /bin/bash 进入 

-i 打开STDIN，用于控制台交互    
-t 分配tty设备, 该可以支持终端登录，默认为false

操作完成 执行exit退出

二、拷贝本地文件到容器中

docker cp 本地文件 /容器中目录位置
docker cp /www/runoob 96f7f14e99ab:/www

保存修改后的docker镜像

docker的镜像的状态是临时，每次启动都会是一个新的，所以我们要保存这个镜像

docker commit -m 'fun' 运行时id 镜像的名称

删除一个镜像

docker rmi 镜像id
